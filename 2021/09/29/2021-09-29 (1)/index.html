  3. 首先，要注册一个GitHub的账号，记住注册名，注册的邮箱和注册密码，非常重要重要。注册成功后，我们要建立一个新的仓库，仓库名称为 “注册名.github.io” 勾选“add a README file”项

  4. 将GitHub与本地绑定。打开“我的电脑”，进入任意一个文件夹，选择“git bush here”

    输入“ssh-keygen -t rsa -C ‘邮件’” 敲四次回车键（为什么是四次？）
    打开C:\Users\Administrator\.ssh

    ctrl a+ctrl c复制其中内容

    然后，打开GitHub的设置界面，点击“SSH and GPG keys”，新建一个SSH keys，题目随意，点击Key栏，ctrl v进行复制

  5. 在本地生成博客的具体内容。新建一个blog文件夹，在文件夹中选择“git bush here” 输入“hexo init”初始化hexo博客

    输入“hexo s”生成本地的hexo页面 浏览器输入http://localhost:4000/进行预览
