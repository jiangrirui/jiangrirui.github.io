int main和void main的区别
void main有返回值 int main没有返回值
void main可以用int main代替
一个函数的函数名既是该函数的代表，也是一个变量。由于函数名变量通常用来把函数的处理结果数据带回给调用函数，即递归调用，所以一般把函数名变量称为返回值（什么意思我也看不懂，好像return 0是返回值）

程序的基本框架
#include <stdio.h>

int main()
{

return 0;

}

“\n”换行

有“\n”
无“\n”

变量是一个保存数据的地方，当我们需要在程序里保存数据时，就需要一个变量来保存它。用一个变量保存了数据，它才能参与到后面的运算中。
变量的名字：标识符，标识符可自行定义
基本原则：标识符只能由字母、数字和下划线组成，数字不可出现在第一个位置上，C语言的关键字不可做为标识符
**=**：赋值；所有变量在使用之前应该赋值一次
变量赋值
给定以下代码段：
int a,b=0;
则a的初始值是0
是不正确的，定义多个变量没办法批量赋值，必须写
int a = 0，b = 0

浮点数带小数点的数值。
浮点：小数点是浮动的，是计算机内表达非整数的一种方式
